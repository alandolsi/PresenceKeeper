# Roadmap

## v0.1.0 (MVP)
- App shell with start/stop controls.
- Configurable interval.
- Countdown display.
- Worker process integration.
- Basic logging and error handling.

## v0.2.0
- Daily schedule engine (start/stop by time).
- Improved validation and user feedback.
- Persisted settings.

## v0.3.0
- System tray mode.
- Optional auto-start on Windows login.
- Enhanced activity/audit logs.

## v1.0.0
- Production hardening.
- Signed Windows installer.
- Full release docs and screenshots.
